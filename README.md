# Mortgage Calculator


**Course:** Mobile Application Development
**Assignment:** Mortgage Calculator Application

## Project Description

This Android application calculates mortgage payments based on the loan amount, interest rate, and loan term selected by the user.

The application contains two screens:

1. Main Screen

   * Displays the mortgage amount
   * Displays the selected loan term
   * Displays the interest rate
   * Calculates and displays the monthly payment
   * Calculates and displays the total payment
   * Includes a Terms and Conditions checkbox
   * Includes a Modify Data button

2. Modify Screen

   * Allows the user to change the mortgage amount
   * Allows the user to select a loan term (10, 15, or 30 years)
   * Allows the user to select an interest rate from a list
   * Returns the updated values to the main screen

## Features

* Mortgage payment calculation
* Monthly payment calculation
* Total payment calculation
* Radio buttons for loan term selection
* ListView for interest rate selection
* Terms and Conditions Alert Dialog
* Data transfer between activities

## Technologies Used

* Android Studio
* Kotlin
* XML Layouts
* Android SDK

## How to Run

1. Clone the repository.
2. Open the project in Android Studio.
3. Allow Gradle to sync.
4. Run the application using an Android Emulator or physical Android device.
